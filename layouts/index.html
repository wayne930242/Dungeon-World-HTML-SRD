Dungeon World HTML SRD

Dungeon World is a tabletop roleplaying game by Sage LaTorra and Adam Koebel of Sage Kobold Productions. This document is a System Reference Document (SRD) for that game, prepared in HTML format for reference in a browser.

This site was designed with the help of Bootstrap and built with Hugo. Source: https://github.com/flyingturtle/Dungeon-World-HTML-SRD.

The original source files for Dungeon World (from which the HTML SRD was derived) can be found on GitHub: https://github.com/Sagelt/Dungeon-World
